# 1.研究動機:現行標準導盲手杖（白手杖）主要透過物理觸碰方式來感知前方地面上的靜態障礙物，例如台階與導盲磚，屬於被動式感知工具。對於人行道中可能快速接近的動態物體，如自行車或跑者，白手杖無法提供即時的事前預警，使視障者往往在發生碰撞後才察覺風險，進而增加行走時的心理壓力與行動不確定性。為改善上述限制，市面上已出現結合超音波感測的電子導盲手杖，可偵測部分懸空障礙物，如突出的招牌或車輛後照鏡。然而，超音波感測在戶外環境中容易受到風切聲與環境噪音干擾，且其波束發散角較大，導致障礙物距離與方位資訊不夠精確，使使用者難以判斷實際危險方向，影響其應用成效。
#
# 2.創新之處:
#   a.全天候軌跡追蹤
<!-- #       痛點： 市售超音波手杖受限於物理特性，易受風切聲干擾，且無法分辨物體是在靠近還是在遠離；影像辨識（AI Camera）則在夜間或強光下失效。 -->
#       採用 24GHz 毫米波雷達 (HLK-LD2450)。
#       抗干擾： 具備穿透雨霧、不懼黑暗與強光的全天候特性。
#       鎖定隱形殺手：利用都卜勒效應，能針對無聲且高速的目標（如斑馬線上的自行車、滑行的電動車）進行精準識別，這是傳統白手杖與超音波完全無法做到的。
#
#   b.動態姿態補償
#       本系統開發了一套感測器融合演算法，即時讀取MPU6050陀螺儀的角速度數據。當偵測到Z軸角速度大於設定閾值（判定為轉身中），系統會自動抑制雷達的測距警報。此機制讓手杖具備了分辨敵我的能力，知道哪些是衝過來的車，哪些是我轉身看到的牆。
#
#   c.TTC碰撞時間預判:面對高速逼近的車輛，系統能在10公尺外提前發出最高級別警報，為視障者爭取關鍵的1.5秒黃金閃避時間。面對慢速行人，則延後警示，減少不必要的心理負擔。
#
#
# 3.實現方法:
#
# 本系統採用模組化設計，核心硬體清單如下：
# 運算核心： Raspberry Pi Pico 2 W。採用 RP2350 雙核心晶片，負責高效能的訊號處理與邏輯運算。
#
# 感知模組：
# HLK-LD2450：負責24GHz雷達波發射與接收，透過UART回傳目標座標(X, Y) 與速度。
# MPU6050：負責偵測手杖的三軸加速度與角速度，透過 I2C 回傳姿態數據。
#
# 回饋模組：
# 震動馬達： 提供不同頻率的觸覺回饋（安全/警戒/危險）。
# 無源蜂鳴器： 提供高頻聲音警示。
# LED燈： 作為對環境的被動警示（呼吸/爆閃），提升夜間行走的可見度 (To Be Seen)。
#
# 4.應用領域:盲人路口穿越輔助(在斑馬線上偵測右轉切入的車輛，提供主動避險資訊) 人行道行走
# 銀髮族安全輔具-針對聽力退化或反應較慢的高齡長者，本系統可整合至拐杖中，作為防止後方追撞的警示裝置。
#
# 5.預期成果:
# 偵測距離與廣度：
#   針對大型金屬目標（如汽車），預期有效偵測距離可達 10~12 公尺，提供使用者約 1.5 秒以上的閃避反應時間。
#   針對中小型目標（如自行車、行人），預期有效偵測距離為 5~8 公尺。
#   偵測角度涵蓋水平 ±60 度，有效消除傳統白手杖僅能探測正前方的盲點限制。
#   誤報率抑制 (False Alarm Reduction)：
#   透過 MPU6050 進行姿態補償，預期能將因使用者轉身、變換方向所產生的環境誤報率降低至 5% 以下（
#   相較於無 IMU 輔助的純雷達系統 ）。
# 系統即時性 (Latency)：
#   基於 Raspberry Pi Pico 2 W 的雙核心邊緣運算，預期從「雷達偵測」到「發出警示」的端到端延遲 (End-to-End Latency) 控制 在 # 100ms (0.1秒) 以內，滿足即時安全需求。
# 功耗與續航：
#   採用單顆LED呼吸燈與休眠演算法優化後，預期在一般使用情境下，裝置續航力可達 6 小時以上，滿足視障者單日外出的需求。
# 
# 功能驗證與應用成效 (Functional & Application Benefits)
# 混合車流中的動態識別：
#   系統能成功利用目標的移動速度與接近率 (TTC)，區分「威脅性目標」與「一般路人」。
#   特別針對斑馬線上的自行車 (U-Bike) 與 路口轉彎車輛，能觸發高頻急促警示，驗證了系統對「無聲高速物體」的主動防護能力。
#   直覺式的人機互動 (HMI)：
#       建立一套漸進式回饋機制：
#           遠距/低威脅： 僅觸發 LED 呼吸燈與微弱震動。
#           近距/高威脅： 觸發高頻蜂鳴聲與紅色爆閃燈。
#       預期此分級機制能讓視障者在不增加心理負擔的前提下，直觀地感知危險等級。
# 雙向安全守護：
#   透過高亮度LED的主動閃爍，預期能提升視障者在夜間或陰雨天的被看見率，警示駕駛人主動減速禮讓。
