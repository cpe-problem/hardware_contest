# Introduction
# pi pico 2w + hlk-LD2450+ mpu6050+震動馬達+蜂鳴器+LED燈+(Grove - Vision AI V2) 
# hlk-LD2450:雷達人體追蹤感測器
# mpu6050:運動追蹤與姿態感測的電子晶片
# 
# 研究目的: 本研究旨在解決視障者在行進間，面臨自行車、路跑者等無聲高速物體突發性逼近的動態感知盲區。
# 整合 (Integrate) 24GHz 毫米波雷達與 MPU6050 慣性測量單元，建構一套適用於 Raspberry Pi Pico 2 W 的低功耗穿戴式硬體架構，以解決傳統輔具無法全天候運作之硬體限制。
#
# 研究動機:現有的標準導盲手杖（白手杖）僅能探測前方地面上的靜止障礙物（如台階、導盲磚），屬於「被動式觸碰」。然而，對於懸空障礙物（如突出的招牌、卡車後照鏡）以及「後方高速逼近的物體」（如自行車、跑者），白手杖完全無法預警。視障者往往是在發生碰撞後才察覺危險，這導致了巨大的心理壓力與行動限制。此外，市售的超音波電子手杖雖然能測距，但在戶外易受風切聲干擾，且超音波發散角過大，難以精準定位物體方位，導致實用性不足。




# 三、 實現方法 (Implementation Method)
# 3.1 硬體架構 (System Architecture)
# 本系統採用模組化設計，核心硬體清單如下：
# 運算核心： Raspberry Pi Pico 2 W。採用 RP2350 雙核心晶片，負責高效能的訊號處理與邏輯運算。
#
# 感知模組：
# HLK-LD2450： 負責 24GHz 雷達波發射與接收，透過 UART 回傳目標座標 (X, Y) 與速度。
# MPU6050： 負責偵測手杖的三軸加速度與角速度，透過 I2C 回傳姿態數據。
#
# 回饋模組 (HMI)：
# 震動馬達： 提供不同頻率的觸覺回饋（安全/警戒/危險）。
# 無源蜂鳴器： 提供高頻聲音警示。
# 高亮度 LED： 作為對環境的被動警示（呼吸/爆閃），提升夜間行走的可見度 (To Be Seen)。
